1. Clone the repository:
   ```bash
   git clone https://github.com/DR-Marques/Testing_E2E_Cypress.git
   cd Testing_E2E_Cypress
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run tests in headless mode:
   ```bash
   npx cypress run
   ```

4. (Optional) Run tests in interactive mode:
   ```bash
   npx cypress open
   ```
